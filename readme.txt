Log variables and view the results in your admin interface

== Description ==

Allows you to log php variables and arrays by using function: `esl_log($var);`.
The Log appears in the admin area.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php esl_log($your_variable); ?>` in your template
